User Requirements

Requirements Brainstorming

As a user, I want to be able to project existing issues from a GitHub repository onto a story map.
As a user, I want to be able to add new categories to the story map to group stories under.
As a user, I want to be able to update existing categories on the story map.
As a user, I want to be able to remove categories from the story map.
As a user, I want to be able to add new stories to the story map, specifying the name, description, category and story points.
As a user, I want to be able to move stories between categories on the story map.
As a user, I want to be able to update existing stories on the story map.
As a user, I want to be able to remove stories from the story map.
As a user, I want to be able to update the issues on GitHub with the changes I made to my local story map.

Activity Monitoring

As a user, I want to be able to view global statistics on the story map stories, such as the number of open stories.
As a user, I want to be able to monitor the state of each story on the story map.
As a user, I want to be able to monitor the urgency of each story on the story map.
As a user, I want to be able to drill-down on particular issues on the story map by clicking on them.
